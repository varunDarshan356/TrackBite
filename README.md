## 📖 Overview

TrackBite is a nutrition tracker that helps users log meals and monitor their daily macronutrient intake.

Whether you're counting calories, tracking protein goals, or simply keeping a record of your meals, TrackBite provides a fast, responsive, and enjoyable experience powered by **React Native**, **Expo**, and **TypeScript**.

---

## ✨ Features

### 🍽 Meal Tracking

* Add meals with custom names
* Log calories, protein, carbohydrates, and fat
* Edit or delete existing meals
* Store meal history locally

### 📊 Nutrition Dashboard

* View daily calorie totals
* Track protein intake
* Monitor carbohydrates
* Track fat consumption
* Instant macro calculations

### 📱 Modern Mobile Experience

* Clean and responsive interface
* Tab-based navigation
* Smooth animations
* Native mobile feel
* Haptic feedback for key interactions

### ⚡ Productivity Features

* View recent meals
* Browse complete meal history
* Copy meal information
* Share meals with others
* Daily reminder notifications 
* Persistent local storage

# ⚙️ Getting Started

## Prerequisites

Before getting started, ensure you have:

* Node.js (LTS)
* npm
* Expo Go (Android/iOS)

or

* Android Studio Emulator
* Xcode Simulator (macOS)

---

## Installation

Clone the repository:

```bash
git clone https://github.com/varunDarshan356/trackbite.git
```

Navigate into the project:

```bash
cd trackbite
```

Install dependencies:

```bash
npm install
```

---

# ▶️ Running the App

Start the development server:

```bash
npx expo start
```

You can then:

* 📱 Scan the QR code with **Expo Go**
* 🤖 Launch an Android emulator
* 🍎 Launch an iOS simulator
* 🌐 Run the web version (if enabled)

---

# 💾 Data Storage

TrackBite stores all meal information locally using **AsyncStorage**.

No account creation or backend service is required.

---

# 🔔 Notifications

The application supports daily reminder notifications using Expo Notifications to encourage consistent meal tracking.
